# Text-Detector
Android OCR App

# Features
<li>
Convert images to text file
</li>
<li>
Editable Text
</li>
<li>
Generate Pdf of text document
</li>
<li>
document management
</li>
<li>
delete
</li>
<li>
document merging
</li>

</li>
<li>
Google Play Services
</li>
<li>
Google Support Library
</li>

</li>
